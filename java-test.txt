Hiding and showing a html?

Sure! Here's a simple FiveM Lua script that shows an HTML UI when a command is used and hides it with another command. This script uses the `NUI` (New User Interface) system in FiveM.

First, create an HTML file named `ui.html` in the `html` folder of your resource:

```html
<!-- html/ui.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>UI</title>
    <style>
        body {
            display: flex;
            justify-content: center;
            align-items: center;
            height: 100vh;
            background-color: rgba(0, 0, 0, 0.5);
            color: white;
            font-size: 24px;
        }
    </style>
</head>
<body>
    <div id="content">Hello, this is your HTML UI!</div>
</body>
</html>
```

Next, create a Lua script named `client.lua` in your resource:

```lua
-- client.lua
local display = false

RegisterCommand("showui", function()
    SetDisplay(true)
end, false)

RegisterCommand("hideui", function()
    SetDisplay(false)
end, false)

RegisterNUICallback("exit", function(data)
    SetDisplay(false)
end)

function SetDisplay(bool)
    display = bool
    SetNuiFocus(bool, bool)
    SendNUIMessage({
        type = "ui",
        status = bool,
    })
end

Citizen.CreateThread(function()
    while true do
        Citizen.Wait(0)
        if display then
            DisableControlAction(0, 1, true) -- LookLeftRight
            DisableControlAction(0, 2, true) -- LookUpDown
            DisableControlAction(0, 142, true) -- MeleeAttackAlternate
            DisableControlAction(0, 18, true) -- Enter
            DisableControlAction(0, 322, true) -- ESC
            DisableControlAction(0, 106, true) -- VehicleMouseControlOverride
        end
    end
end)
```

Finally, create a `__resource.lua` or `fxmanifest.lua` file to define your resource:

```lua
-- __resource.lua
resource_manifest_version '44febabe-d386-4d18-afbe-5e627f4af937'

client_script 'client.lua'

ui_page 'html/ui.html'

files {
    'html/ui.html'
}
```

Or if you're using `fxmanifest.lua`:

```lua
-- fxmanifest.lua
fx_version 'cerulean'
game 'gta5'

client_script 'client.lua'

ui_page 'html/ui.html'

files {
    'html/ui.html'
}
```

With this setup, you can use the `/showui` command to display the HTML UI and the `/hideui` command to hide it. Let me know if you need any more help!
